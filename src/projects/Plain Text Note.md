
Plain Text Note
===============

A set of tools to create plain text note.  

1. Quick Launcher 2  
2. A markdown converter (Coming soon...)  

* Why I Use Plain Text Notes  

Plain text notes provide a way for note-takers to bypass concerns about platform compatibility, software bugs, application startup times, software updates, and server connections. The only requirement is a text editor, a feature available on all computers.  

Taking notes should be about concentrating on the content, not the process. Once a note is saved (to git), it is preserved forever. Over the years, plain text notes have become my primary method for taking notes, and now I have hundreds of these text-based notes.  

* Why I Don't Use Markdown  

Markdown has its merits, particularly for those who need to share their notes online via HTML. A lots of tools can do that.  
But, you still need to type two spaces every line end.  
You still need to take care of typo miss because it has its strick format.  
When you copied a URL, you need to add `()` before and after.  
...  
And that's too much work.
