
Plain Text Note
===============

A set of tools to create plain text note.  

1. [Quick Launcher 2](https://github.com/lhypds/quick-launcher-2)  
2. A markdown converter  

* Why I Use Plain Text Notes  

Plain text notes let note-takers forget about platform compatibility,  
software bugs, software starting, software updating, server connection...  
The only requirement, is a text editor, and its available on all computers.  

Note takers focus on the note, once a note is saved (to git), it is preserved forever.  
You can easily sync it with iCloud, Dropbox, Google Drive...  
Over the years, plain text notes have become my primary method for taking notes,  
and I created hundreds of these text-based notes.  

Although, there is Markdown, particularly for those who need to share their notes online via HTML.  
But, you still need to type two spaces every line end.  
You still need to take care of typo miss because it has its strict format.  
...
